# flutter_demo

Flutter demo 工程

## 项目说明

+ 编译工程
  ```
  flutter build apk     编译 apk
  flutter build ios     编译 ios
  flutter build web     编译 web 版本
  flutter pub get       安装依赖包
  flutter devices       设备列表
  
  ```

+ vs code 插件配置
  + flutter
  + dart
  + Awesome Flutter Snippets
  + Pubspec Assist
  + bloc 
  + Dart Data Class Generator
